What is Literal in Java?
A literal is a fixed value directly written in the code.
When you write a value directly in your program, that value is called a literal.


What is SCP (String Constant Pool) in Java?
SCP (String Constant Pool) is a special memory area inside the Heap where Java stores string literals to save memory.
Java does not create duplicate string objects if the value is the same.
It reuses the existing string from SCP.


Why SCP is Used?
Save memory
Improve performance
Avoid duplicate objects


note : SCP is a memory area where Java stores unique string literals
