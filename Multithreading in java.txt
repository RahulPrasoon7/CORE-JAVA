Multithreading :  

Multi threading  is an process where we can executes multiple threads at simultaneously / same times 


Thread : 

Thread is small piece of code : 
Collection of Thread is called as Process :
Thread is also an class present in java.lang package 
in Thread class we have methods like as CurrentThread , start , run ... 

multi threading can be achieved by two ways : 
trough 

1> Thread class 
2> Runnable Interface 


to create an thread we have two methods which is provided by Thread class 

1> run     
2> start


1> Functional Interface : 

--> it contains only one abstract method
--> it has Annotated base as (@FunctionalInterface)
Ex : Runnable Interface 

2> Marker Interface :

--> it has Empty Interface
--> it doesn't contains any methods or (any Members)
Ex : Serializable , Cloneable , RandomAccess etc 





How we can create Thread : 

Firstly we need to create child of Thread class 
used methods --> 1. run (thread logic) , 2. start(to start Thread)

IMP Note : for run method execution used start method : 



ref code : 




package com.thread;

public class MyThread extends Thread {

	
	public void run() {

		for (int i = 1; i <= 100; i++) {
			System.out.println("myThread " + i);
			try {
				Thread.sleep(500);
			} catch (InterruptedException e) {
				e.printStackTrace();
			}
		}
	}

	public void m1() {
		for (int i = 111; i <= 200; i++) {
			System.out.println("m1 logic  " + i);
			try {
				Thread.sleep(400);
			} catch (InterruptedException e) {
				e.printStackTrace();
			}
		}

	}

}



 




package com.thread;

public class MainClass {

	public static void main(String[] args) {

		MyThread t = new MyThread();
		t.start(); // it calls run method
		t.m1();

	}

}
